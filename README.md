## Algorithm-Solutions

### 진행 방법

- 돌아가면서 매주 한 명씩 알고리즘 문제를 골라온다.
- 매주 월요일부터 새 문제를 시작하며, 그 주 일요일 23시 59분까지 각자 풀이 commit 후 Pull Request 한다.
- 그 다음 주 화요일 23시 59분까지 다른 사람들의 풀이를 보고 코드 리뷰를 달아준다.
